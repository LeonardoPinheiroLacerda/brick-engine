== Architectural Purpose

The `GameScore` module acts as the strict authority dedicated to tracking real-time player performance metrics, levels, and point multipliers. It prevents transient session logic from interacting with persistent storage, explicitly taking over saving the global `highScore` into an isolated `LocalStorage` namespace directly, ensuring different game engines do not overwrite each other's records.

It exposes reactivity through an event-based system using `xref:../enums/ScoreProperty.adoc[ScoreProperty]`.

=== Score Management

Methods for adding and resetting points.

==== `score: number`

Read-only property returning the current session's points.

'''

==== `increaseScore(amount: number): void`

Adds points to the current session.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`amount` |`number` |Base raw integer points to add before multiplier application.
|===

**Behavior**

Increases the score by `amount * multiplier`. Automatically compares the new score against the internal `highScore` property and updates it if a new record is reached (which in turn persists the new record to `LocalStorage`). Triggers a `SCORE` notification via the `xref:EventEmitter.adoc[EventEmitter]`.

'''

==== `resetScore(): void`

Resets the session score to zero. Triggers a `SCORE` notification.

'''

==== `highScore: number`

Read and Write property for the highest overall score achieved.

**Behavior**

When set, it automatically persists the value directly to `LocalStorage` under the namespaced key `${gameId}::highScore`. This ensures that different games do not overwrite each other's records. Triggers a `HIGH_SCORE` notification.


'''

=== Level Management

Methods for difficulty scaling.

==== `level: number`

Read-only property returning the current difficulty level.

'''

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`amount` |`number` |Positive integer count of step levels to advance globally.

**Behavior**

Increases the internal level and triggers a `LEVEL` notification.

'''

==== `resetLevel(): void`

Resets the level counter to 1. Triggers a `LEVEL` notification.

'''

==== `maxLevel: number`

Read and Write property for the maximum game level cap. Triggers a `MAX_LEVEL` notification on change.
