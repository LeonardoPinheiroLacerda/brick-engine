The `HudRenderer` manages the visualization of the Head-Up Display, including scores, levels, system status, and the specialized HUD grid. It implements the standard `Renderer` interface and is managed by the `GameRenderer`.

== API Reference

=== `setup(rendererMetrics: RendererMetrics): void`

Pre-calculates static HUD geometry.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`rendererMetrics` |`RendererMetrics` |The layout metrics calculated by `GameRenderer`.
|===

**Behavior**

Calculates and caches the absolute pixel positions for HUD borders and grid cells. This ensures that the HUD remains perfectly aligned regardless of the display resolution.


'''

=== `render(grid: Cell[][], modules: GameModules): void`

Main HUD drawing orchestration.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`grid` |`Cell[][]` |The main game grid (unused, but required by interface).
|`modules` |`GameModules` |Access to all modules (Text, Score, State, etc.) for dynamic data extraction.
|===

**Behavior**

1.  Clears the HUD area.
2.  Renders the pre-cached static borders.
3.  Draws the dynamic content (current score, high score, level, and game state messages).
4.  Renders the 4x4 HUD grid state.
