The `RendererContext` is a singleton utility class used to provide centralized, global access to the P5.js instance (`p5`) throughout the Brick Engine.

== Purpose

By acting as a central provider, `RendererContext` eliminates the need to pass the `p5` object via dependency injection (constructor arguments) to every component, view, layout, and module in the architecture. It significantly reduces boilerplate code and prevents tightly coupling components to the rendering library from the top down.

== API Reference

=== `init(p: p5): void`

Initializes the context with the active P5.js instance.

**Parameters**

[cols="1,1,3", options="header"]
|===
|Name |Type |Description
|`p` |`p5` |The P5.js instance initialized by the engine's entry point.
|===

**Behavior**

This method captures the `p5` instance and stores it internally for global access. It includes a safeguard: trying to call this method if the context is already initialized will output a warning and will not overwrite the existing instance.


'''

=== `reset(): void`

Clears the currently stored P5.js instance.

**Behavior**

This is an internal utility primarily used for unit testing. It ensures test isolation by wiping the context between tests when virtual P5 mock instances are created and torn down in `beforeEach` closures.


'''

=== `p`

Getter property to access the initialized P5.js instance.

**Returns**

The initialized P5.js instance.

**Behavior**

Returns the internal reference to the `p5` instance. In standard project operation, it is safe to access. However, accessing this getter before `init()` is called will result in a runtime error, preventing silent failures.

== General Usage

To access P5.js functions globally, components use ES6 object destructuring against `RendererContext`:

[source,typescript]
----
import RendererContext from '../context/RendererContext';

class MyButton {
    draw() {
        // Destructure 'p' from the context
        const { p } = RendererContext;
        
        // Draw the button
        p.fill(255);
        p.rect(0, 0, 10, 10);
    }
}
----
