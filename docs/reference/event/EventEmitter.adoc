The `EventEmitter` is a static helper class serving as a simple, global Publish-Subscribe (pub/sub) bus for the Brick Engine. It allows decoupled communication between different parts of the system or client games.

== API Reference

=== `subscribe<T>(eventName: string, callback: EventCallback<T>): void`

Subscribes a listener to an event.

**Parameters**

[cols="1,1,3", options="header"]
|===
|Name |Type |Description
|`eventName` |`string` |The identifier for the event.
|`callback` |`Function` |The function to execute when the event is emitted.
|===

**Behavior**

Registers the callback function in an internal array mapped to `eventName`.

'''

=== `unsubscribe<T>(eventName: string, callback: EventCallback<T>): void`

Unsubscribes a listener from an event.

**Parameters**

[cols="1,1,3", options="header"]
|===
|Name |Type |Description
|`eventName` |`string` |The identifier for the event.
|`callback` |`Function` |The exact function reference to remove.
|===

**Behavior**

Removes the specific callback from the event's listener array. Fails silently if the event or callback is not registered.

'''

=== `notify<T>(eventName: string, payload?: T): void`

Emits an event, triggering all registered callbacks.

**Parameters**

[cols="1,1,3", options="header"]
|===
|Name |Type |Description
|`eventName` |`string` |The identifier for the event to emit.
|`payload` |`T` (optional) |Data to pass to the callbacks.
|===

**Behavior**

Synchronously iterates over and executes all callbacks registered for the specified `eventName`, passing the optional payload to each.

'''

=== `clear(): void`

Clears all registered events and listeners.

**Behavior**

Wipes the internal event registry. This is primarily useful for cleanup operations, such as tearing down state during unit tests.
