== Architectural Purpose

The `Time` interface defines the core contract for managing the engine's logical clock and update frequency. It ensures that any implementation provides the necessary tools to decouple deterministic game logic from the visual frame rate.

== API Reference

=== Core Simulation

==== `update(deltaTime: number): void`

Advances the internal physical stats (FPS/TPS).

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`deltaTime` |`number` |Time physically elapsed since the last rendering frame in milliseconds.
|===

'''

==== `accumulate(deltaTime: number): void`

Adds elapsed time to the simulation accumulator.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`deltaTime` |`number` |Time in milliseconds to add to the simulation.
|===

'''

==== `shouldTick(): boolean`

Evaluates if a logical simulation tick should be executed.

**Returns**

`boolean` - `true` if at least one tick was consumed.

'''

==== `reset(): void`

Resets all internal timers and counters.

'''

=== Declarative API

==== `atInterval(ms: number): boolean`

Checks if the logical clock has crossed a millisecond interval.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`ms` |`number` |The millisecond interval to check.
|===

**Returns**

`boolean` - `true` if the interval boundary was crossed.

'''

==== `every(ms: number, action: () => void): void`

Executes a callback every N milliseconds of logical game time.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`ms` |`number` |Millisecond interval.
|`action` |`() => void` |Callback function.
|===

== General Usage

Implemented by `xref:../../modules/GameTime.adoc[GameTime]`. This interface allows other modules to depend on a standardized timing contract without being tied to a specific implementation.
