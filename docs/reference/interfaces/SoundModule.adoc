= SoundModule Reference

Interface for triggering audio effects and managing volume.

== Properties

[cols="1,1,3"]
|===
|Name |Type |Description
|`muted` |`boolean` |The current audio suppression state.
|===

== Methods

=== `play(sound: Sound): Promise<void>`
Triggers an asynchronous playback of the specified audio asset.

'''

=== `stop(sound: Sound): Promise<void>`
Abruptly terminates the playback of a specific sound identifier.

'''

=== `stopAll(): Promise<void>`
Stops all active audio channels.

'''

=== `toggleMute(): void`
Flips the current `muted` property.

== General Usage

Implemented by `GameSound`. It abstracts the Web Audio API complexities, allowing games to simply call `modules.sound.play(Sound.DROP)`.
