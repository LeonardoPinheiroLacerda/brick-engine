= StateSyncable Reference

Contract for modules that must react to global state changes.

== Properties

[cols="1,1,3"]
|===
|Name |Type |Description
|`_state` |`State` |Internal reference to the state module.
|===

== Methods

=== `syncState(state: State): void`
Links the module instance and allows it to subscribe to properties.

== General Usage

Implemented by `GameSound` and `GameScore`. During the start-up sequence, the engine calls `syncState()` on these modules to ensure they have the latest references to `muted` flags and `highScore` data.
