Enumeration of observable properties within the `xref:../modules/GameScore.adoc[GameScore]` module, used primarily for event subscriptions via `xref:../modules/EventEmitter.adoc[EventEmitter]`.

== Attributes

[cols="1,3"]
|===
|Name |Description
|`SCORE` |The total session sequence points.
|`LEVEL` |The active difficulty scaling level.
|`MULTIPLIER` |The current point application multiplier.
|`MAX_LEVEL` |The ceiling for level progression.
|`HIGH_SCORE` |The persistent record value.
|===

== General Usage

These values act as the event keys when calling `subscribe` or `unsubscribe` on the `xref:../../interfaces/modules/Score.adoc[Score]` interface.
