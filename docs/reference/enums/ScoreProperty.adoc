Enumeration of observable properties within the `xref:../modules/GameScore.adoc[GameScore]` module, used primarily for event subscriptions via `xref:../modules/EventEmitter.adoc[EventEmitter]`.

== Values

[cols="1,1,3", options="header"]
|===
|Name |Type |Description
|`SCORE` |`string` |The total session sequence points.
|`LEVEL` |`string` |The active difficulty scaling level.
|`MULTIPLIER` |`string` |The current point application multiplier.
|`MAX_LEVEL` |`string` |The ceiling for level progression.
|`HIGH_SCORE` |`string` |The persistent record value.
|===

== General Usage

These values act as the internal event keys. Public subscriptions to score changes should use the simplified `subscribe` method on the `xref:../../interfaces/modules/Score.adoc[Score]` interface, which internally utilizes `ScoreProperty.SCORE`.
