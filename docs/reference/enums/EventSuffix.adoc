== Architectural Purpose

The `EventSuffix` enum defines the available suffixes used to scope events within the engine's contextual notification system.

== Values

[cols="1,3"]
|===
|Value |Description
|`PLAYING` |The game is actively running.
|`PAUSED` |The game is started but currently halted.
|`TITLE` |The engine is powered on but the game has not started yet.
|`GAMEOVER` |The current game session has ended.
|===

== General Usage

These suffixes are used by the `xref:../modules/EventEmitter.adoc[EventEmitter]` to construct specialized event names (e.g., `UP:pressed:playing`) and by modules like `xref:../modules/GameControl.adoc[GameControl]` and `xref:../modules/GameState.adoc[GameState]` for state-scoped subscriptions.
