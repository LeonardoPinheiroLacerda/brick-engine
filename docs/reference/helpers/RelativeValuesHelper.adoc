== Architectural Purpose

The `RelativeValuesHelper` provides isolated mathematical layout functions specifically designed to scale absolute sizes to the base runtime canvas. It guarantees isolated calculations for responsive dimensional generation dynamically without hardcoding absolute `p5` rendering parameters.

== Methods

=== `getRelativeWidth(size: number): number`

Calculates a pixel width based on a normalized percentage.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`size` |`number` |The desired scale fraction bounded between 0.0 and 1.0 representing percentage of width.
|===

**Returns**

`number` - The calculated absolute pixel width.
