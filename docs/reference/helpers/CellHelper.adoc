== Architectural Purpose

The `CellHelper` provides a reliable and centralized mechanism to instantiate default grid cells. It ensures that every cell generated across the engine instances starts with identical baseline properties, preventing inconsistent matrix states from forming in abstract spaces.

== Methods

=== `emptyCell(coordinate: xref:../types/Coordinate.adoc[Coordinate]): xref:../types/Cell.adoc[Cell]`

Creates a new cell with default "OFF" state values.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`coordinate` |xref:../types/Coordinate.adoc[Coordinate] |The initialized precise `{x, y}` logical grid position to assign to the new cell instance.
|===

**Returns**

`Cell` - A cell object with `value: 0` and `xref:../enums/Color.adoc[Color].DEFAULT`.

**Behavior**

This is used primarily by `xref:../modules/GameGrid.adoc[GameGrid]` to populate the initial matrix during allocation or when resetting a row/column.
