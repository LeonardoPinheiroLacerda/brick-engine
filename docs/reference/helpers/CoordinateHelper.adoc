== Architectural Purpose

The `CoordinateHelper` acts as the engine's translation layer between the different abstract mathematical models (e.g., the logical `xref:../modules/GameGrid.adoc[Game Grid]` or scaling normalized values from 0.0 to 1.0) and the absolute physical layout on screen. It ensures consistency in calculating element sizes dynamically so interfaces remain strictly responsive to the runtime canvas resolution.

== API Reference

=== `getRelativeCoordinate(coordinate: xref:../types/Coordinate.adoc[Coordinate]): xref:../types/Coordinate.adoc[Coordinate]`

Converts a pixel-based coordinate into a normalized relative coordinate (0.0 to 1.0).

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`coordinate` |xref:../types/Coordinate.adoc[Coordinate] |The absolute pixel coordinate.
|===

**Returns**

`xref:../types/Coordinate.adoc[Coordinate]` - The new coordinate structure scaled proportionately to the physical base canvas width/height.


'''

=== `getDisplayPosX(x: number, displayWidth: number): number`

Calculates the absolute X pixel position on the main game display.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`x` |`number` |The relative or normalized input X horizontal position (0.0 to 1.0).
|`displayWidth` |`number` |The dynamically calculated width of the central display area.
|===

**Returns**

`number` - The absolute horizontal pixel position.


'''

=== `getDisplayPosY(y: number, displayHeight: number): number`

Calculates the absolute Y pixel position on the main game display.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`y` |`number` |The relative or normalized input Y vertical position (0.0 to 1.0).
|`displayHeight` |`number` |The dynamically calculated height of the central display area.
|===

**Returns**

`number` - The absolute vertical pixel position.


'''

=== `getHudPosX(x: number, displayWidth: number): number`

Calculates the absolute X pixel position within the HUD area.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`x` |`number` |The relative or normalized X position specifically within the bounded HUD compartment.
|`displayWidth` |`number` |The total real width of the central display (used for zero-point alignment offset calculation).
|===

**Returns**

`number` - The absolute horizontal pixel position within the HUD.


'''

=== `getHudPosY(y: number, displayHeight: number): number`

Calculates the absolute Y pixel position within the HUD area.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`y` |`number` |The relative or normalized Y position specifically within the bounded HUD compartment.
|`displayHeight` |`number` |The absolute pixel height assigned logically to the central display (used for scaled positioning bounds).
|===

**Returns**

`number` - The absolute vertical pixel position within the HUD.
