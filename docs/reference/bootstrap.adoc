== Architectural Purpose

The `bootstrap.ts` module acts as the absolute architectural start sequence isolating complex viewport interactions from gameplay bounds. It encapsulates the hardware-dependent framework injection (P5.js canvas creation) and constructs the active Context pointers before explicitly deferring the draw loops seamlessly back onto the Client's `Game` constructor instance.

== API Reference

=== `bootstrap(ClientGame: ClientGameConstructor): p5`

Orchestrates structural point of initialization isolating strict client code execution.

|===
