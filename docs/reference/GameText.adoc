= GameText Module
:toc: macro
:toc-title: Table of Contents
:icons: font
:source-highlighter: highlight.js

toc::[]

The `GameText` module handles rendering text on the game screen. It specifically manages retro-style fonts and ensures text is scaled correctly relative to the canvas size.

== Areas

There are two main areas for text:

. **Display**: The main game grid area.
. **HUD**: The Head-Up Display area (usually on the side).

== Font Sizes

The engine provides responsive font sizes via the `FontSize` enum:

* `EXTRA_SMALL`
* `SMALL`
* `MEDIUM`
* `LARGE`
* `EXTRA_LARGE`

== API Methods

=== Configuration

* `setTextSize(fontSize: FontSize)`: Sets the current font size.
* `setTextAlign(align: FontAlign, verticalAlign: FontVerticalAlign)`: Sets alignment (CENTER, LEFT, RIGHT, TOP, BOTTOM).
* `setActiveText()` / `setInactiveText()`: Sets the color to the theme's active or inactive color.

=== Rendering

* `textOnDisplay(text: string, coordinate: Coordinate)`: Renders text on the main display. Coordinates are normalized (0 to 1).
* `textOnHud(text: string, coordinate: Coordinate)`: Renders text on the HUD. Coordinates are relative to the HUD grid.

=== Example

[source,typescript]
----
const { text } = this.modules;

// Render "SCORE" on the HUD
text.setTextSize(FontSize.SMALL);
text.setTextAlign(FontAlign.LEFT, FontVerticalAlign.TOP);
text.setActiveText();
text.textOnHud('SCORE', { x: 0, y: 0 });
----
