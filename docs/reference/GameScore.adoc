= GameScore Module Reference
:toc: macro
:toc-title: Table of Contents
:icons: font
:source-highlighter: highlight.js

toc::[]

The `GameScore` module tracks player performance during a session. It manages the current score, game levels, and multipliers, while ensuring high scores are correctly validated and persisted via `GameState`.

== API Reference

=== `score: number`

Read-only property returning the current session's points.


'''

=== `level: number`

Read-only property returning the current difficulty level.


'''

=== `multiplier: number`

Read and Write property for the current points multiplier.


'''

=== `increaseScore(amount: number): void`

Adds points to the current session.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`amount` |`number` |Base points to add.
|===

**Behavior**

Increases the score by `amount * multiplier`. Automatically compares the new score against the persisted `highScore` in `GameState` and updates it if a new record is reached.


'''

=== `resetScore(): void`

Resets the session score to zero.


'''

=== `increaseLevel(amount: number): void`

Increases the game difficulty level.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`amount` |`number` |Number of levels to add.
|===


'''

=== `resetLevel(): void`

Resets the level counter to 1.


'''

=== `getFormattedScore(digits: number): string`

Returns the score as a padded string suitable for HUD display.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`digits` |`number` |Total length of the string (e.g., 6 for "000150").
|===

**Returns**

`string` - The zero-padded score string.


'''

=== `syncState(state: State): void`

Links the score module with the persistent game state.

**Parameters**

[cols="1,1,3"]
|===
|Name |Type |Description
|`state` |`State` |The `GameState` module.
|===

**Behavior**

Implements the `StateSyncable` interface. This connection is used to fetch and update the global high score.


'''

=== `getDebugData(): Record<string, string | number | boolean>`

Returns metadata for the real-time debugger.
