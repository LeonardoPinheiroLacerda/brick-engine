= Vector Reference

Represents a generic 2D vector for directional or magnitude calculations.

== Attributes

[cols="1,1,3"]
|===
|Name |Type |Description
|`x` |`number` |Horizontal offset.
|`y` |`number` |Vertical offset.
|===

== General Usage

Unlike `Coordinate`, which usually implies a position, `Vector` is used for transformations, velocities, and offsets applied to coordinates.
